Theme Changer JS

This is a simple theme changer JS package to allow changing theme to and from dark mode.

Elements:

This package is composed of 4 files:

  • theme-changer.js (using minifier.org theme-changer-min.js)
  • theme-changer.css (using minifier.org theme-changer-min.css)
  • light-icon.svg
  • dark-icon.svg

Usage:

To use this package add to the header of your page the following line: 

      
        <link rel="stylesheet" href="_html/theme-changer.css" />

Or on the css: 

      
        @import url('theme-changer.css');

On the html place the following lines where you want it the theme changer to appear: 

      
        <div class="theme-toggle"></div>
        <script src="_html/theme-changer.js"></script>

On the page's css you must define the following variables (check example.css and example.html for full example): 

      
        :root {
          /*example css*/
          --background-color: #f6f6f6;
          --background-color-section: #ccc4;
          --font-color: #222;
          
          /*theme-changer vars*/
          --tc-foreground-color: var(--font-color);
          --tc-background-color: var(--background-color);
          --tc-on-background-color: var(--background-color-section);
        }
